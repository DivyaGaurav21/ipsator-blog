# Ipsator-Blog Application

 ## Objectives:
  - Use Stytch for password less user authentication. 
  - Use Sanity CMS for storing and fetching blog posts. 
  - Use GA4 for analytics (user-engagement).
  - Use Tailwind css for styling


### Tech stack Used : -
  - Next Js 
  - Sanity CMS 
  - Stych for user - auth 
  - Tailwind CSS  
  - JS  
  - GA4
